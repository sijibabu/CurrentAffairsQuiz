# Current Affairs Quiz (Java)

## Overview
A GUI-based Current Affairs Quiz application developed using Java Swing. The application presents multiple-choice questions, records user responses, and displays the final score through an interactive graphical interface.
## Features
- Multiple-choice quiz questions
- Interactive user interface
- Score calculation
- Simple and user-friendly design
- Developed using Java

## Technologies Used
- Java
- Java Swing 

## How to Run
1. Download or clone this repository.
2. Open the project in VS Code or any Java IDE.
3. Compile the Java file:
   ```
   javac CurrentAffairsQuiz2025.java
   ```
4. Run the program:
   ```
   java CurrentAffairsQuiz2025
   ```
